## Short launch post

Weekaroo v0.1.0 is live.

It is a local-first wall dashboard for the week ahead: ICS calendars, weather, notes, timers, countdowns, and optional AI-generated family messages on hardware you control.

Built for kitchen displays, classroom boards, roommates, and Raspberry Pi kiosks.

https://github.com/Fins1600/weekaroo/releases/tag/v0.1.0

## Contextual reply version

If you are looking for a local calendar/weather wall display, I built Weekaroo for exactly that use case. It runs on your own machine, uses local config files, supports ICS calendars, and has demo data so you can try it without connecting anything private.

https://github.com/Fins1600/weekaroo/releases/tag/v0.1.0
