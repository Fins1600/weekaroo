# Weekaroo brand assets

Weekaroo now has a simple launch-ready identity: a calendar badge with a kangaroo silhouette, plus a clean wordmark for README, release notes, and social previews.

## Files

- `assets/logo.svg`, primary wordmark for GitHub README and docs
- `assets/logo.png`, PNG fallback for social platforms or previews that do not render SVG
- `assets/logo-mark.svg`, square mark for avatars, icons, and compact placements
- `assets/logo-mark.png`, PNG fallback for avatars and social profiles
- `docs/images/weekaroo-social-card.svg`, editable 1280x640 social preview card
- `docs/images/weekaroo-social-card.png`, post-ready social preview card

## Usage

Use the full logo when there is enough horizontal space, especially GitHub README, release notes, launch emails, and longer posts.

Use the mark for avatars, favicons, small preview thumbnails, or places where the wordmark would be too small.

Use the social card for launch posts, GitHub social previews, and link attachments.
